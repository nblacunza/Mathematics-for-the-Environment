Iterative Progress:
2D Newton-Raphson Method

Visual monitoring of the convergence path towards the system root. Each figure represents an iteration where the Jacobian matrix linearizes the nonlinear field.

Mathematical Basis

The update rule for a system of equations $F(\mathbf{x}) = 0$ is defined as:

$\mathbf{x}_{n+1} = \mathbf{x}_n - J_F(\mathbf{x}_n)^{-1} F(\mathbf{x}_n)$

Where $J_F$ is the Jacobian matrix of the system.
