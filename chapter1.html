Dichotomy

A robust bracketing method based on the Intermediate Value Theorem. It ensures convergence if the function changes sign across the interval.

Secant

Utilizes a linear approximation between two points. It offers superlinear convergence without the need for analytical derivatives.

Newton

The gold standard for local convergence. It uses the function's derivative to achieve quadratic speed toward the root.
