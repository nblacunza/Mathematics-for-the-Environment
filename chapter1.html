Solving Nonlinear Equations

Numerical analysis is the study of algorithms that use numerical approximation for the problems of mathematical analysis. This chapter focuses on finding roots of equations where an exact algebraic solution is impossible—a fundamental challenge when modeling the Earth's energy balance.

"The characters of the universe are written in the language of mathematics." Explore how we translate these characters into Julia code.

Why Julia?

Numerical Methods Approach

In environmental sciences, we often deal with "equilibria"—states where the net energy change is zero. Mathematically, this means solving $f(x) = 0$. We start with simple interval-halving and move towards high-speed iterative shoots.

Didactic Note: Every algorithm presented here balances three key factors: robustness (will it find the root?), speed (how many steps?), and requirements (do we need the derivative?).

Iterative Methods

Dichotomy

Based on the Intermediate Value Theorem. It's the most reliable method: if the function changes sign, you will find the root.

Secant Method

An approximation that uses two points to draw a line. Faster than dichotomy and doesn't require calculating derivatives analytically.

Newton-Raphson

The gold standard for speed. By using the tangent at a point, it achieves quadratic convergence toward the solution.

Julia Implementation

Fixed-Point Iteration (Picard) 
function picard_iteration(g, x0; tol=1e-7, max_iter=100)
    x = x0
    for i in 1:max_iter
        x_new = g(x)
        if abs(x_new - x) < tol
            return x_new, i
        end
        x = x_new
    end
    error("Did not converge")
end
2D Newton-Raphson Base 
function newton_2d(f, J, x_start; tol=1e-8)
    x = x_start
    while norm(f(x)) > tol
        x -= J(x) \ f(x) # Efficient solver
    end
    return x
end
