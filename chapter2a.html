Elementary Topology

Before diving into complex dynamics, we must define the "space" where our models live. Topology provides the rigorous framework to discuss limits, continuity, and the proximity of environmental states in multidimensional systems.

"Mathematics is the art of giving the same name to different things." Explore the structures of metric spaces and norms in Julia.

Why Julia?

The Geometry of Stability

To understand if a climate equilibrium is "stable", we need to define what it means to be "near" that equilibrium. This chapter reviews the topology of the real line and extends it to general metric spaces, setting the stage for studying global attractors.

Didactic Note: Metrics and norms are not just abstract definitions; they are the yardsticks we use to measure the error in our numerical simulations and the convergence of our algorithms.

Julia Implementation

Calculating Vector Norms 
using LinearAlgebra

x = [3.0, -4.0, 0.0]

# Standard Euclidean Norm (L2)
norm_l2 = norm(x) 

# Manhattan Norm (L1)
norm_l1 = norm(x, 1)

# Infinity Norm (Max)
norm_inf = norm(x, Inf)
Custom Metric Definition 
# Define a custom distance function
function environmental_dist(u, v, weight)
    # Weighted Euclidean distance
    return sqrt(sum(weight .* (u .- v).^2))
end

dist = environmental_dist([20, 10], [21, 11], [1.0, 0.5])
