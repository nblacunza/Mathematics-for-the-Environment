In the context of modern environmental modeling, mathematics is the language, but execution requires a tool capable of translating theory into high-performance simulations. The choice of Julia for the numerical problems in this work is a strategic response to the needs of contemporary scientific computing.

Why Julia?

Julia was designed to solve the "two-language problem," offering a syntax as intuitive as Python with the execution speed of Fortran or C. For the study of geophysical fluids and non-linear dynamics, this balance is essential.

What is Pluto.jl?

Pluto is a reactive notebook environment for Julia, designed for scientific computing, teaching, and reproducible research. Unlike traditional notebooks, Pluto notebooks are built around the idea that results should always reflect the current state of the code.

In Pluto, notebooks behave like reactive documents: whenever a variable or function is modified, all dependent computations are automatically updated. This eliminates hidden state, reduces errors, and makes the logical structure of numerical experiments explicit.

Throughout this book, Pluto notebooks will be the primary computational medium for exploring models, visualizing dynamics, and experimenting with parameters. Each numerical problem is designed to be executed, modified, and extended interactively.

  • P1 Reactive by design: Outputs are always consistent with the code, preventing hidden dependencies.
  • P2 Reproducible: Each notebook contains its own package environment, ensuring long-term reproducibility.
  • P3 Exploratory: Ideal for parameter studies, bifurcation exploration, and dynamical systems intuition.

Why this workflow?

The combination of Julia and Pluto allows us to treat numerical experiments not as black-box simulations, but as transparent mathematical objects. Code, equations, and results coexist in a single, coherent document.

  • 01 Fast: Julia compiles to efficient native code via LLVM.
  • 02 Dynamic: Interactive exploration without sacrificing performance.
  • 03 Reproducible: Fully controlled computational environments.
